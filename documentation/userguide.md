# Käyttöohje

## Ohjelman käynnistäminen

Käännetty ohjelmatiedosto sijaitsee projektin alihakemistossa: ```application/build/libs/```. Ohjelma voidaan käynnistetään komennolla:
```
java -jar application/build/libs/FocusStacking.jar
```
